## 1. PURPOSE

This document provides bidirectional cross-reference mapping between the Design History File (DHF) sections of the CardioGuard™ ICD Design History File and the Technical Sections of the Pre-Market Approval (PMA) application submitted per 21 CFR Part 814. The mapping ensures that:

1. Every DHF document has a clear path to the PMA submission where it is cited as evidence.
2. Every PMA Technical Section requirement can be traced back to specific DHF evidence documents.
3. No regulatory requirement is left without supporting documentation.
4. FDA reviewers can efficiently locate supporting evidence referenced in PMA Technical Sections.
