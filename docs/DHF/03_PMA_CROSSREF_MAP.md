## 1. Purpose

This document provides the bidirectional cross-reference between the Design History File (DHF) sections and documents for the CardioSync CRT-D Model CS-9000 and the technical sections of PMA Application P240001, submitted pursuant to 21 CFR Part 814. The cross-reference matrix enables FDA reviewers to identify the DHF source documents supporting each PMA technical section, and enables NovaCor to confirm that all PMA technical sections are adequately supported by DHF documentation.

This map also cross-references applicable FDA guidance documents and ISO/IEC consensus standards for each mapped area.
